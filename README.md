## Featured project

### [config-sentinel](https://github.com/raj-singh-giift/config-sentinel) &nbsp; [![npm](https://img.shields.io/npm/v/config-sentinel?style=flat)](https://www.npmjs.com/package/config-sentinel) [![CI](https://github.com/raj-singh-giift/config-sentinel/actions/workflows/ci.yml/badge.svg)](https://github.com/raj-singh-giift/config-sentinel/actions)

Catch missing `process.env` keys **before they reach production** — static AST analysis for Node.js.

```bash
npx config-sentinel ./src
```

Zero runtime overhead · Works with `dotenv`, `convict`, raw `process.env` · TypeScript-native
